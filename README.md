# RavenDB Java Client

Official RavenDB Java Client

Simply put ravendb dependency to your pom:

```
<dependency>
  <groupId>net.ravendb</groupId>
  <artifactId>ravendb</artifactId>
  <version>4.0.0-rc2</version>
</dependency>
```
