# RavenDB Java Client

Offical RavenDB Java Client

Simply put ravendb dependency to your pom:

```
<dependency>
  <groupId>net.ravendb</groupId>
  <artifactId>ravendb-client</artifactId>
  <version>4.0.0-rc1</version>
</dependency>
```
